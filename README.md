## 🛠️ Hybrid Multi-Language Core Architecture

AetherGuard separates high-velocity transaction handling from complex semantic parsing loops via a decoupled microservices mesh:

1.  **The High-Velocity Gatekeeper (Go 1.26):** Intercepts Model Context Protocol (MCP) and Agent-to-Agent (A2A) payload traffic at wire-speed. Resolves basic structural bounds (Identity validations, Action matching matrices, and FinOps transaction budget limits) in microseconds.
2.  **The Isolated Execution Vault (Python 3.11):** Dynamically extracts abstract code or deep-reasoning properties. Utilizes Pythonic **Abstract Syntax Tree (AST)** node compilation to test strings within a secure bubble, instantly dropping persuasion-bombing and shell-escapes.
3.  **The Shared Persistent Registry (SQLite):** Decouples security configurations from source code. Policies update on the fly inside a self-contained local storage ledger, mapping structured variables across all microservices simultaneously.
4.  **The Live Command Interface (HTML5/CSS3 Web Panel):** Provides centralized visibility, active telemetry, and a complete time-series compliance log framework for administrative audits.

🚀 Rapid Local Deployment
Prerequisites
Docker Desktop installed and active.

Local network access to open ports 8080, 5000, and 3000.

1. Provision & Initialize Container Mesh
Clone this repository and spin up the complete integrated multi-language stack with a single command:

Bash
docker compose down -v
docker compose up --build
Docker will compile the binary executables, establish isolated environment sandboxes, mount the shared SQLite volume at /app/db/aetherguard.db, and stitch communications channels together.

2. Verify Management Viewports
Open your web browser and target the administrative command console:
👉 http://localhost:3000
